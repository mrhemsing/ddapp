# Dark Drives Voice Reference

Source repo: `../darkdrives`

Use the sales site's established mixed samples as the voice/mix reference for generated tour audio:

- `public/audio/dark-drives-hero-sample-mixed.mp3`
- `public/audio/dark-drives-teaser-sample-mixed.mp3`

Transcripts are copied beside the audio in `public/audio/*.txt` and captions in `*.vtt`.

Tone notes from the source material:

- Direct, second-person, in the car.
- Short declarative lines.
- Real-place framing: "Here. On this ground."
- Suspense from restraint, not jokes or jump-scare copy.
- Safety/legal text stays plain and unromanticized.

Do not commit ElevenLabs API keys. When real generation is added, read `ELEVENLABS_API_KEY` from local environment only.
